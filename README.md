Stickonomics

Stickonomics is a satirical web app that simulates the Indian stick economy and LPG cylinder subsidy prediction, blending humor with lightweight
data storytelling. Built with React on the frontend and Node.js on the backend, it demonstrates a modern full-stack structure while keeping the UI
simple and interactive. Users adjust parameters such as income, usage, political view, and festival behavior to observe 'probability' outputs and
reflect on fuel subsidy narrative in a tongue-in-cheek manner. The app connects front-end controls to an Express-based API and offers fallback
behavior when external services are unavailable.
